```
Perform a thorough code review of this [LANGUAGE] code.

CODE TO REVIEW:
[PASTE CODE]

REVIEW CATEGORIES:

1. FUNCTIONAL CORRECTNESS
- Does it do what it's supposed to?
- Are edge cases handled?
- Are there obvious bugs?
- Error handling in place?

2. CODE QUALITY
- Readability and maintainability
- DRY principles followed?
- Function/method length appropriate?
- Naming conventions consistent?

3. PERFORMANCE
- Time complexity reasonable?
- Memory usage appropriate?
- Any N+1 queries?
- Unnecessary computations?

4. SECURITY
- Input validation present?
- SQL injection prevention?
- XSS prevention?
- Authentication/authorization checks?
- Secrets handling?

5. BEST PRACTICES
- Language idioms followed?
- Design patterns appropriate?
- Testing considerations?
- Documentation needed?

6. ERROR HANDLING
- Exceptions caught appropriately?
- Logging in place?
- User-friendly error messages?
- Graceful degradation?

OUTPUT FORMAT:
For each issue found:
- Severity: [CRITICAL/HIGH/MEDIUM/LOW]
- Line number (if applicable)
- Issue description
- Suggested fix with code

SUMMARY:
- Total issues: [COUNT]
- Critical: [X]
- High: [X]
- Medium: [X]
- Low: [X]
```

# SECTION 5: PROMPT FRAMEWORKS

## Framework 1: The RACE Prompt Method
```
Use this framework for any task:

R - Role: [WHO YOU ARE]
A - Action: [WHAT TO DO]
C - Context: [BACKGROUND/DETAILS]
E - Expectation: [OUTPUT FORMAT]

Example:
Role: Expert SEO copywriter
Action: Write a product page
Context: Eco-friendly water bottle, target outdoor enthusiasts, premium positioning
Expectation: SEO-optimized copy, 800 words, conversion-focused, H2/H3 structure
```

## Framework 2: The SCOPE Prompt Method
```
S - Situation: [CURRENT STATE]
C - Complication: [PROBLEM]
O - Objective: [GOAL]
P - Plan: [APPROACH]
E - Execution: [SPECIFICS]

Use for problem-solving and strategy.
```

## Framework 3: The PEEL Prompt Method
```
P - Point: [MAIN IDEA]
E - Evidence: [SUPPORT/DATA]
E - Explanation: [ANALYSIS]
L - Link: [CONNECT TO BROADER CONTEXT]

Use for analytical content and arguments.
```

## Framework 4: The AIDA Prompt Structure
```
A - Attention: [HOOK]
I - Interest: [DETAILS]
D - Desire: [BENEFITS]
A - Action: [CTA]

Use for marketing and sales content.
```
