# Content Creation Prompts

## 1. Blog Post Generator
```
Create a comprehensive blog post about [TOPIC]. 

Include:
- A compelling headline
- An engaging introduction (2-3 paragraphs)
- At least 5 main sections with H2 headings
- Practical tips readers can implement
- A strong conclusion with CTA

Target audience: [AUDIENCE]
Word count: [NUMBER] words
Tone: [TONE - professional/friendly/humorous]
SEO keywords to naturally incorporate: [KEYWORDS]
```
