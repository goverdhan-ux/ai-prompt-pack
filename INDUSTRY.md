## Prompt Engineering Special

### 1. Chain of Thought
```
Think through this problem step by step:

[PROBLEM/QUESTION]

Show your reasoning at each step. Consider:
- Initial analysis
- Assumptions
- Alternative approaches
- Potential issues
- Final conclusion
```

### 2. Few-Shot Learning Example
```
Here are examples of [TYPE] done correctly:

Example 1:
[EXAMPLE 1]

Example 2:
[EXAMPLE 2]

Now create [NEW REQUEST]:

[REQUEST]

Follow the same pattern.
```

### 3. Role-Play Scenario
```
I want you to role-play [SCENARIO].

You play: [CHARACTER/ROLE]
I play: [CHARACTER/ROLE]
Setting: [CONTEXT]

Start the conversation naturally. Stay in character until I say "stop."

[ADD ANY SPECIFIC REQUIREMENTS]
```

### 4. Socratic Questioning
```
Help me think through this decision:

[DECISION/SITUATION]

Use Socratic questioning:
- Ask clarifying questions
- Challenge assumptions
- Explore alternatives
- Consider implications
- Guide to conclusion (don't give it)

Ask one question at a time and wait for my response.
```

### 5. Output Refinement
```
Based on this output:

[PASTE OUTPUT]

Refine it by:
- Improving clarity
- Enhancing engagement
- Adding specific details
- Adjusting tone
- Strengthening the call-to-action

Show both original and refined versions.
```
